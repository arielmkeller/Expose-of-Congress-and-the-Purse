# Join Columns

Use these columns for fuzzy or exact joins to USAspending data:
- `recipient_clean`: normalized recipient name
- `project_clean`: normalized project description or purpose
- `member_clean`: normalized member name
- `amount_requested`: numeric amount for range or exact matching
- `fiscal_year`: fiscal year alignment

`project_id` is reserved for a future deterministic ID once a stable scheme is chosen.
